## Core Game Mechanics (2026.04.09 장기투자 방향으로 재설계)
1. **프리시즌 리서치** - 종목 카드(기업 개요, 재무, 섹터, 실적) 열람 + 워치리스트 작성
2. **Snake Draft** - 종목 순서대로 선택, 실시간 포트폴리오 분산도 피드백 제공
3. **6개월 시즌 리그** - 주간 H2H가 아닌 시즌 누적 성적으로 순위 경쟁
4. **제한적 트레이드** - 월 2~3회만 허용, 잦은 매매 페널티 → 신중한 의사결정 ���련
5. **투자 리포트** - 매월 자동 생성, 포트폴리오 성과 회고 + 투자 스타일 객관화
6. **시즌 결산 & 플레이오프** - 장기 성과 기반 최종 우승자 결정

### 장기 vs 단타 비교 모드 (부가 기능)
- 동일 드래프트에서 출발, 단타 리그는 자유 매매 가능
- 시즌 결산 시 장기 리그 vs 단타 리그 다차원 비교 (수익률, 거래 횟수, 최대 하락폭, 수익 효율)
- "같은 출발점, 다른 운용 방식, 다른 결과" → 데이터가 스스로 가르치는 교육
- 단타가 수익률에서 이기더라도, 리스크 조정 수익률(수익 효율, 최대 하락 등)에서 장기가 우위로 설계

## Scoring System (2026.04.09 장기 지표 중심으로 재설계)
| 지표 | 의미 | 교육 효과 |
|------|------|-----------|
| 누적 수익률 | 시즌 동안의 총 수익 | 장기 성과 추적 |
| 수익 효율 점수 | 위험 대비 수익 효율 (내부적으로 샤프비율) | 리스크 관리 의식 |
| 최대 하락 방어력 | 최고점 대비 최대 하락 (내부적으로 MDD) | 하락장 방어 전략 |
| 포트폴리오 분산도 | 섹터·종목 분산 수준 | 분산투자의 가치 체득 |
| 배당수익�� | 안정적 배당 수익 기여도 | 인컴 투자 이해 |
| 섹터 상대수익 | 같은 섹터 대비 초과 성과 | 종목 선택 능력 |

**설계 원칙:** "안정적으로 잘 운용한 사람이 이기는" 구조. 유저 대면 용어는 쉬운 말 사용 (샤프비율→수익 효율 점수, MDD→최대 하락 방어력)

- **B2B 장점**: 학교/교육청 단위 계약 = 낮은 CAC, 높은 리텐션 (매년 반복 사용)
- **모두의 창업 심사**: "에듀테크"가 "MZ 투자자 게임"보다 심사위원에 설득력 높음
- **사회적 가치**: 금융 문해력 격차 해소 어필 가능

### K-12 타겟의 리스크
- **판매 사이클 느림**: 교육기관 의사결정 6~12개월, 교과서 연계/교사 연수/시범 운영 필요
- **게임성 vs 교육성 충돌**: 둘 다 중간만 하는 제품이 될 위험
- **TAM 축소**: 성인 1,423만 → K-12 약 530만 (초중고), 실질 TAM 더 작음
- **글로벌 확장 어려움**: 각국 교육과정/규제 상이

### 잠정 전략: 듀얼 트랙
```
Core Product: StockWise League (성인 대상, 판타지리그) → 본업, 수익 엔진, 글로벌 스케일
Education Mode: StockWise Academy (K-12 대상, 교육기관 납품) → 동일 엔진 + 교육 래핑
  - 가상 종목, 학습목표 매핑, 교사 대시보드, 성취도 리포트
```
- 초기 수익은 B2C(성인), 교육 시장은 중기 확장으로 로드맵에 배치

### 교육기관 납품 시 필요 요소

## 미성년자 이용 법적 검토 (리서치 완료 2026.04.08)

### 결론: 미성년자 이용 법적으로 가능 (금지 아님)
StockWise League는 실제 돈을 투자하지 않으므로 자본시장법 적용 대상이 아니며, 현금 보상이 없으면 사행성 게임에도 해당하지 않음.

### 법률별 분석
1. **자본시장법 — 해당 없음**
   - 금융투자상품 정의(제3조): 실제 금전 손실 위험이 있는 권리 → 시뮬레이션은 해당 없음
   - 투자권유(제9조 4항): 실제 매매를 권유하지 않으므로 해당 없음
2. **게임산업진흥법 — 등급 심사 필요**
   - GRAC 등급분류 필수 (전체이용가 또는 12세이용가 예상)
   - 현금/현금성 보상 없으면 사행성 게임 아님
3. **개인정보보호법 — 14세 미만 부모 동의 필수**
   - 14세 미만: 개인정보 수집 + 게임 계정 생성 모두 법정대리인 동의 필요
   - 14~17세: 게임 계정 생성 시 법정대리인 동의 + 실명/연령 인증
   - 18세 이상: 일반 절차

### 선례 (미성년자 모의투자 이미 활발)
- **토스 틴즈**: 만 7~18세 모의투자 서비스 정식 운영, 모의투자 대회 개최
- **청소년 모의주식투자대회**: 신한투자증권 + 퍼핀 주관, 14~19세 대상, 3회째 운영
- **한국청소년투자대회**: koreayic.org에서 별도 운영
- **미성년자 실제 증권 계좌**: 토스증권 만 0세부터 부모 동의로 개설 가능, 14세+ 본인 매매 가능
- → 실제 주식 매매도 가능한데 시뮬레이션 게임이 안 될 이유 없음

### StockWise League 준수 요건 3가지
1. 게임 결과에 현금/현금성 보상 없을 것 → 사행성 회피
2. 실명/연령 인증 + 미성년자 법정대리인 동의 플로우 구현
3. GRAC 등급 심사 통과 (전체이용가/12세이용가 목표)
